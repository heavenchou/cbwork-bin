CBETA XML TEI P4 => P5 Steps

Ray Chou 2007.1.8

X2R TXT => XML

  • CVS update \cbwork\common\X2R
  • cd \cbwork\bin\p5
  • 執行 x2r-xml.py

gaiji-m.mdb => cbgaiji.xml

MS Access 開啟 gaiji-m.mdb 匯出 XML 到 /cbwork/bin/p5/cbgaiji.xml

p4=>p5

  • 修改 p5.py 堛爾纁|參數
  • 執行 p5.py 一次產生一冊, 或執行 p5.bat 一次多冊.
  • 使用 7z.py 壓縮, 上傳給標記組

比對以下兩個版本

  • p5 xml => txt: 執行 cbwork/bin/p5/p5totxt-dir.py
  • p4 xml => txt: xml2txt -h -z -x 1 (-h: 不要檔頭資訊, -z: 不使用通用字, -x 1: 悉曇字使用 entity &SD-xxxx;)
