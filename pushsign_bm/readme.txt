BM 簡單標記版標點插入程式  by heaven                2019/02/19

使用方法：

pushsign_bm.pl 標點版.txt 原始的BM.txt 成果檔BM.txt > 記錄檔.txt


標點版.txt => 就是已經標好新式標點的文件。

原始的BM.txt => 沒有標點的簡單標記檔。

成果檔BM.txt => 產生的結果檔，理論上就是標好新標的簡單標記檔。

記錄檔.txt => 一些檢查用的記錄檔

==================================

【已知問題】

假設標點版為

如是我聞，一時....

原始的BM為

如是[你問>我聞]一時

則無法順利比對下去。要先處理標點版，也做成修訂格式，例如先改成

如是[你問>我聞]，一時....

就可以比對下去了。

但如果修訂只有一個字，則只要 [A>B] A 或 B 某一字和新標版相同就會過關。

==================================

【已知問題】

假設標點版為

如是我聞「一時....

原始的BM為

如是我聞[01]一時

則產生結果為

如是我聞[01]「一時

但也許正確的是

如是我聞「[01]一時

所以這些可能要再檢查。

[01] 也可能是其他簡單標記，如 <p>。
