BM 簡單標記版標點插入程式  by heaven                2019/02/19

有二支程式要處理

使用方法：

perl modi_corr.pl 標點版.txt 原始的BM.txt 新的標點版.txt

perl pushsign_bm.pl 新的標點版.txt 原始的BM.txt 成果檔BM.txt


標點版.txt => 就是已經標好新式標點的文件。

原始的BM.txt => 沒有標點的簡單標記檔。

新的標點版.txt => 有處理修訂格式的新式標點版。

成果檔BM.txt => 產生的結果檔，理論上就是標好新標的簡單標記檔。

記錄檔.txt => 一些檢查用的記錄檔

==================================

【說明】

假設標點版為

如是我聞，一時....

原始的BM為

如是[你問>我聞]一時

這樣無法直接處理。

所以先使用 modi_corr.pl 程式，產生新的標點版，內容為：

如是[你問>我聞]，一時....

再用這個版本和 BM 比對即可。


==================================

【已知問題】

假設標點版為

如是我聞「一時....

原始的BM為

如是我聞[01]一時

則產生結果為

如是我聞[01]「一時

但也許正確的是

如是我聞「[01]一時

所以這些可能要再檢查。

[01] 也可能是其他簡單標記，如 <p>。
